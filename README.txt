## Features

### Deck
- **Red Deck**: Gives 10 discards.
- **Blue Deck**: Gives 10 hands.
- **Yellow Deck**: Gives a 999,999,999 cash.
- **Joker Slots**: Every deck now provides 45 joker slots.

### Jokers
- Every Joker is OP now lol (yes, all 150 jokers)
- **Joker Abilities**:
  Joker abilities that are related to chance will be guaranteed (except for debuff).

## Features

### Deck
- **Red Deck**:  
  +10 discard every round

- **Blue Deck**:  
  +10 hand every round  

- **Yellow Deck**:  
  Start with an extra $999,999,999

- **Green Deck**:  
  You don't earn interest. Instead, gain $5 per remaining Hand and $5 per remaining Discard at the end of each round.  

- **Black Deck**:  
  +45 Joker slot, but -1 hand every round  

- **Magic Deck**:  
  Start run with the Crystal Ball voucher and 2 copies of The Fool.  

- **Nebula Deck**:  
  Start run with the Telescope voucher but -1 consumable slot.  

- **Ghost Deck**:  
  Spectral Cards may appear individually in the shop, and you start with a Hex card.  

- **Abandoned Deck**:  
  Start run with no Face Cards (Jacks, Queens or Kings) in your deck.  

- **Checkered Deck**:  
  Start run with 26 Spades and 26 Hearts in deck, and no Clubs or Diamonds.  

- **Zodiac Deck**:  
  Start the run with Tarot Merchant, Planet Merchant, and Overstock vouchers.  

- **Painted Deck**:  
  +5 Hand Size, -0 Joker Slot.  

- **Anaglyph Deck**:  
  After defeating each Boss Blind, gain a Double Tag.  

- **Plasma Deck**:  
  Balance Chips and Mult when calculating score for played hand. X2 base Blind size. (Your chips and mult become averaged before scoring)  

- **Erratic Deck**:  
  All Ranks and Suits in deck are randomized.

**NOTE**:
- Every desk gives 45 Joker slot

### Enhanced Cards
- **Bonus Card**: +9999999999 Bonus Chips
- **Mult Card**: +999,999,999,999,999 Mult
- **Wild Card**: Still the same but with +999,999,999 Mult
- **Glass Card**: 99x Mult, 1 in 100000000000 Chance to break
- **Steel Card**: 99x Mult while this card is held in hand
- **Stone Card**: +9999999999999 Bonus Chip
- **Gold Card**: -1000 Cash if this card is held in hand until the end of a round
- **Lucky Card**: 1 in 5 chance for +777 Mult, 1 in 15 chance to win $777
